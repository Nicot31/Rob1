# Rob1

This project is based on a STM32F407EVT6 Dark board with a TFT LCD ILI9341 that include 
a touch screen based on XPT2046.

![alt text](https://github.com/Nicot31/Rob1/blob/master/pictures/STM32.png)
![alt text](https://github.com/Nicot31/Rob1/blob/master/pictures/LCD.png)

## Description
Based on FreeRTOS, this program manage a TFT LCD with a touch screen. 

It is edited with the STM32CubeIDE framewoark. The Core part is generated with STM32CubeMX.
